# Instagram Profile Analysis

A mini data analysis project built using Python.

## Project Overview

This project analyzes Instagram profile data collected from approximately 40 Instagram accounts.

The raw data is processed and converted into structured dictionaries for analysis.

## Features

- Read Instagram profile data from a text file
- Parse and clean profile information
- Store data using Python dictionaries
- Find the account with maximum posts
- Find the account with maximum followers
- Identify different profile categories

## Technologies Used

- Python
- Lists
- Dictionaries
- File Handling

## Learning Outcomes

- Data Parsing
- Data Cleaning
- Python Data Structures
- Basic Data Analysis
